# Logs Naming Convention

Use this filename format for run logs:

`YYYY-MM-DD_<run-type>.md`

Examples:
- `2026-05-02_full-rebuild.md`
- `2026-05-03_resume-run.md`
- `2026-05-03_cleanup.md`

Guidelines:
- One file per major operation.
- Use lowercase kebab-case for `<run-type>`.
- Keep a short summary, metrics, verification notes, and next actions.
